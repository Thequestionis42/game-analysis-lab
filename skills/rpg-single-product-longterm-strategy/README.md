# RPG-like 单产品长线版本策略分析 Skill

本 Skill 用于针对单个 RPG-like 游戏生成长线版本策略分析 HTML 报告。

## 使用顺序

1. 先阅读 `SKILL.md`。
2. 在正式分析前，必须阅读 `examples/GO_GO_MUFFIN_PRODUCT_UNDERSTANDING_EXAMPLE.md` 作为产品理解层的质量校准范例。
3. 使用 `templates/INPUT_TEMPLATE.md` 整理输入。
4. 在生成 HTML 前，必须阅读 `examples/HTML_REPORT_STRUCTURE_EXAMPLE.md` 作为报告结构和组件形态范例。
5. 按 `templates/REPORT_OUTLINE.md` 生成 HTML 报告。
6. 交付前执行 `templates/PRE_DELIVERY_AUDIT.md`。

## 范例说明

范例只用于校准分析结构、深度和证据意识。不得机械套用《出发吧麦芬》的产品结论到其他游戏。
