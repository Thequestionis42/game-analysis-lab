# game-analysis-lab

Game product research, live-ops analysis, version history studies, and reusable analysis workflows.

## Contents

- `skills/game-update-history-research/`  
  Codex Skill for analyzing game version update histories with public-source validation and producing decision-ready HTML reports.

## Research Scope

This repository is intended for reusable game analysis workflows, including:

- version log research
- mainline level/chapter cadence analysis
- progression-system evolution
- live-ops, activity, season, guild, co-op, social, and economy mechanism analysis
- source-backed HTML reports and templates
